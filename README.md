
# AI-Driven Solutions for Sustainable Energy & Non-Profit Automation

This repository contains two primary AI implementations developed as a technical assessment for an internship position. 

## Project 1: Micro-Climate Aware Solar Power Prediction
An explainable machine learning pipeline utilizing a Random Forest Regressor to forecast commercial solar array outputs. 

### Key Features
* **Feature Engineering:** Novel implementation of rolling 3-step solar irradiance volatility tracking to account for fast cloud transitions.
* **Thermal Loss Analytics:** Custom module-to-ambient temperature differential calculations to proxy semiconductor performance degradation.
* **Accuracy Achieved:** R2 Score of 94.8% on unseen validation datasets.

---

## Project 2: Multilingual Support Engine for Resource-Constrained NGOs
An interactive, multi-step automated customer service pipeline designed to optimize communication channels for non-profits.

### Key Features
* **Intent Classification Step:** Automatically routes incoming queries into Donor, Beneficiary, or General categories.
* **Dynamic Translation:** Replies organically in the user's input language.
* **Session Memory Feature:** Retains dialogue context to handle organic follow-up questions cleanly.
